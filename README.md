# url-shortener
A quick, light-weight, URL shortener built with Node,js

#Usage:
Please download nodejs from [NodeJS](https://nodejs.org).

Please download Express library:
``` npm install express ```

In the console, run:
``` node script.js ```

Open up your browser and type in:
``` http://localhost:4000 ```


Enjoy the shortener!!

**Languages:** JavaScript, EJS.
